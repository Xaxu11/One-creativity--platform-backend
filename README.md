OneCreativity Platform Backend

Project Description

OneCreativity is a creative service platform that connects clients, freelancers, and studios. This backend project demonstrates how a Node.js and Express application communicates with a PostgreSQL database.

Technologies Used

Node.js
Express.js
PostgreSQL
pg
dotenv

Features

Create a service
Get all services
Get a service by ID
PostgreSQL database integration

API Endpoints

Create Service

POST /services

Get All Services

GET /services

Get Service By ID

GET /services/:id
